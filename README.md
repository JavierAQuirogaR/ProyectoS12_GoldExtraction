# ProyectoS12_GoldExtraction
Optimización en el proceso de extracción del oro

## Propósito
Realizar predicciones con distintos modelos e hiperconfiguraciones para determinar cuál modelo y configuración es la adecuada para optimizar el proceso.
## DataSet
Se cuenta con los elementos químicos utilizados en el proceso así como las cantidades usadas y el resultado al finalizar el proceso para cada caso.
## Resultados
Se determinan las variables que son suceptibles de mejora y los parámetros del modelo para aplicar una mejora en el proceso.
